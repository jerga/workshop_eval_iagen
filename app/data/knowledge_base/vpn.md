# VPN - Connexion et depannage

## Symptomes frequents

- La connexion VPN echoue avec un message d'authentification.
- La connexion coupe apres quelques minutes.

## Procedure

1. Verifiez votre connexion internet locale.
2. Ouvrez le client VPN officiel de l'entreprise.
3. Saisissez votre identifiant professionnel.
4. Validez la demande MFA sur votre application mobile.
5. Si la connexion echoue, redemarrez le client VPN puis reessayez.

## Escalade

Si le probleme persiste au-dela de 15 minutes, ouvrez un ticket categorie `network/vpn`.
