# Impression - Imprimantes reseau

## Procedure

1. Verifiez que l'imprimante apparait `en ligne`.
2. Relancez le service d'impression local.
3. Supprimez les travaux bloques puis relancez l'impression.

## Cas frequent

Si l'imprimante n'apparait pas, reinstallez-la depuis le portail IT.

## Escalade

Si le probleme persiste, ouvrez un ticket `workplace/printing` avec le numero de l'imprimante.
