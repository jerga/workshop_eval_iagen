# Messagerie - Probleme d'envoi/reception

## Verification rapide

1. Verifiez votre connexion internet.
2. Fermez puis rouvrez le client de messagerie.
3. Verifiez l'espace disponible de votre boite.

## Si les emails restent en file d'attente

- Verifiez que le mode hors ligne est desactive.
- Reauthentifiez votre compte.

## Escalade

Si l'incident concerne plusieurs utilisateurs, ouvrez un ticket `messaging/incident`.
