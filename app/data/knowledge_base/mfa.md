# MFA - Activation et verification

## Activation

1. Connectez-vous au portail de securite interne.
2. Scannez le QR code avec l'application MFA approuvee.
3. Saisissez le code a 6 chiffres pour confirmer l'association.

## Bonnes pratiques

- Ne partagez jamais vos codes MFA.
- Activez une methode de secours si disponible.

## Probleme courant

Si vous changez de telephone, reinitialisez MFA via le portail libre-service.
