# Onboarding poste de travail

## Jour d'arrivee

1. Recuperer le poste et les acces aupres de l'IT desk.
2. Changer le mot de passe temporaire.
3. Activer MFA.
4. Verifier l'acces email, intranet et VPN.

## Applications standard

- Suite bureautique
- Client VPN
- Agent antivirus
- Outil de ticketing

## Assistance

En cas de blocage, ouvrir un ticket `workplace/onboarding`.
