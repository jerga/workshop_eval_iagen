# Preparation d'un nouveau laptop

## Checklist rapide

1. Mettre a jour le systeme d'exploitation.
2. Installer les applications obligatoires.
3. Configurer le chiffrement disque.
4. Activer l'antivirus.
5. Verifier la connexion VPN.

## Validation

Le poste est pret une fois les acces email, intranet et outils collaboratifs confirmes.
